## Project Setup

feel free to skip this if you already know how to setup project locally and know how to do a PR
<Steps>
1. Fork the Repository
2. Clone your fork
3. Execute the command

   For installing proejct depedency

   ```
   pnpm install
   ```

   Running project

   ```
   pnpm run dev
   ```

   Building project

   ```
   pnpm run build
   ```

4. Make your own branch
   Example:
   ```
   git checkout -b <username>/<feature>
   ```
5. Commit your changes and push it to forked repo
6. Make a PR
</Steps>
